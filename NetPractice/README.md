# Net Practice

## Description

The **main goal** of "Net Practice" project is to get introduced to the basics of **computer networking**. Project is about configuring small-scale networks.

## Instructions

1. Read the README.md file to understand the fundamental concepts for the project.
2. Open the simulated network environment:
	* download the file attached to the project’s page (intra.42.fr)
	* extract the files into any folder of your choice
	* run `run.sh` file like this: `./run.sh`
	* choose the _**training**_ interface for practicing 
	* choose the _**evaluation**_ interface for evaluation 
3. In the training section you can download your configuration by clicking on "Get my config" block
4. It is required to submit .json configuration files for all 10 levels

#### Concept usages in short

	* OSI (open systems interconnection), which is developed by ISO (International Organization for Standardization), is a reference model used to understand networking through 7 layers of abstraction.
	* TCP/IP is the real-world model that implements concepts similar to OSI.
	* TCP/IP addressing (IP addresses) is used to uniquely identify devices in a network.
	* Subnet masks are defining network and host boundaries.
	* Subnetting allows dividing a network into smaller sub-networks.
	* Default gateways allow us to communicate outside the local area network (LAN).
	* Routing tables determine how packets are forwarded between networks.
	* Routers and switches are networking devices used to connect hosts and forward traffic within and between networks.
		‣ Routers forward packets between networks, while switches connect devices within the same local network.
